## About
Foundry is my personal portfolio website showcasing my projects, blogs, and photography.

## Overview
The website is built with a combination of:
* **HTML/CSS/JavaScript** for frontend structure and interactivity.
* **Folder structure**
    * `assets/` - images, styles, scripts
    * `projects/` - project pages
    * `gallery/` - photo gallery
    * `blog/` - blog posts

## Features
* Responsive portfolio layout
* Interactive project and gallery pages
* Intergrated blog section
* Simple compact form

## Notes
